홈 게임 생활툴

빠른 안내

게임은 학습/연습 목적의 모의 기능이며, 투자 조언이 아닙니다.

차트게임

사용법

1) 종목/구간을 선택 후 시작합니다.
2) 매수/매도 버튼으로 거래합니다.
3) 결과는 “게임 종료(정산)”에서 확인합니다.

팁 버튼이 반응하지 않으면 입력값(수량/비율)을 확인한 뒤 다시 시도하십시오.
미니게임

3종 안내

- 캔들 패턴 퀴즈: 패턴을 보고 정답을 선택합니다.
- 지지·저항: 차트를 클릭해 선을 찍고 점수를 확인합니다.
- 반응속도 매매: 조건이 나오면 빠르게 버튼을 눌러 점수를 얻습니다.

생활툴

계산기 안내

각 계산기는 입력값을 기반으로 결과를 즉시 계산합니다. 숫자 입력이 어려우시면 예시 값을 넣고 확인 후 수정하십시오.
