AdSense 연결 메모
1) index.html <head>의 script src에 ca-pub-XXXXXXXXXXXXXXXX 를 본인 ca-pub-번호로 교체
2) 광고 자리(ins.adsbygoogle)의 data-ad-client / data-ad-slot 도 AdSense에서 받은 값으로 교체
3) 광고는 1개만 넣어두었습니다(과도한 광고 자제).

거래 규칙(최종):
- 매수: 현금의 10/30/50/100% 비중으로 즉시 → 현재 '종가'에 체결
- 매도: 보유수량의 10/30/50/100% 비중으로 즉시 → 현재 '종가'에 체결

거래 규칙(업데이트):
- 즉시 없음
- 매수/매도 버튼 클릭 시 '현재 종가(close)' 기준 즉시 체결
- 매수: 현금의 10/30/50/100% 비중
- 매도: 보유수량의 10/30/50/100% 비중

추가 반영:
- 수량 입력 매수/매도도 즉시 없이 '현재 종가' 즉시 체결

광고 제거: index.html의 AdSense 스크립트/광고 영역/관련 CSS 삭제 완료
